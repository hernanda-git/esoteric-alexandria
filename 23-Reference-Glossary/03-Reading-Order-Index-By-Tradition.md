# Reading Order — Index by Tradition

## How to Use This Index

The library has **24 categories** with **60+ documents**. The index is **organized by tradition** so the practitioner can find the **relevant documents** for the tradition that **calls** to them.

The reading order is **not** a single path. The reading order is **5 paths** (the 5 major tradition-families), and the practitioner can choose the path that **calls**.

## The Meta-Reading Order

For the **thorough** practitioner, the **12-month curriculum** (see [[22-Case-Studies-Curricula/01-12-Month-Curriculum]]) is the **recommended path**. The 12-month curriculum covers **all** the major traditions in **one year**.

## The Note

The 5 paths are **suggestions**. The practitioner can read in **any** order. The library is designed to support **multiple** reading orders, not **one** linear path.

The library is a **network**, not a **sequence**. The cross-links (`[[wikilinks]]`) connect the documents in **all directions**. The practitioner can **start anywhere** and **go in any direction**.
