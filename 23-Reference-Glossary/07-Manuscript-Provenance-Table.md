# Manuscript Provenance Table — Codex to Print to Digital

## Why Provenance Is a Reference Tool

The **manuscript trajectory** of a text is not just **historical theater**: it changes which words, diagrams, and ritual formulas reached later magicians. When a compiler in the **17th century** copied a **13th-century** ritual, he may have **merged** it with a different planetary hour table, a **corrupted name**, or a **marginal gloss** promoted to main text. The modern editor then decides which version becomes canonical.

This table tracks the **known witnesses** for the **principal operative texts** in the library, the **transmission news** between them, and the **recommended modern edition**. It is the philological companion to [[05-Source-Index-Major-Texts]].

## Transmission Pathology Index

The most consequential **collisions** between texts:

1. **Picatrix attribution drift** — Later Latin copies attribute planetary images to *Ramon Llull* or *Artephius* to increase prestige; modern editors restore the Arabic source labels.
2. **Ars Notoria + Goetia graft** — The notary prayers appear inside some Goetia variants, blurring the boundary between "meditative" and "conjuring" texts.
3. **Lemegeton title accretion** — Because *Lemegeton* simply means "compendium", later catalogues confused it with *Grimorium Verum*, *The Grand Grimoire*, and other Solomonic wrappers.
4. **Agrippa image drift** — The 1533 editions vary in sigil engraving; modern reprints choose one lineage, leaving the others out of circulation.

## The Practice

A systematic researcher can:

1. **Collect edition notes** from all modern editions cited in [[05-Source-Index-Major-Texts]] and map them to the table above.
2. **Rewrite problematic citations** where a document cites only the popular edition without acknowledging variant MSS.
3. **Maintain provenance links** from individual docs in `12-Agrippa-Picatrix-Grimoires` back to this reference so future authors do not reinvent manuscript claims.
